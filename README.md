### 1. No Personal Data Collection
This dashboard is a **fully client-side application**. 
* **No Backend / No Database:** We do not host a server, and we do not store, track, or collect any of your personal data or search queries.
* **Local Caching:** To optimize performance and bypass GitHub API rate limits, this application uses your browser's local storage (`localStorage`) to temporarily cache repository data for a period of time. This data stays entirely on your local device and is never transmitted to us.

* **Hosting (GitHub Pages):** This website is hosted by GitHub, Inc. (88 Colin P. Kelly Jr. St, San Francisco, CA 94107, USA). * **Data Source (GitHub API):** To display trending repositories, your browser sends direct, unauthenticated requests to the official GitHub API (`api.github.com`). GitHub may process your IP address and connection data during these technical requests to prevent abuse.
